# 🎰 中国城市旅游轮盘

一个有趣的、随机推荐中国旅游城市的 Web 应用

## ✨ 功能特点

- 🎯 **转盘抽取**：通过精美的转盘动画随机选择旅游城市
- 🔍 **智能筛选**：支持按地区和类型筛选城市
- 📍 **详细信息**：展示城市景点、美食、最佳季节等信息
- 📱 **响应式设计**：完美适配桌面和移动设备
- 🎨 **精美动画**：流畅的转盘旋转和结果展示动画

## 📊 已收录城市

当前已收录 **20个** 中国热门旅游城市，包括：

- **华东**：杭州、上海、苏州、南京、厦门、青岛、黄山
- **华北**：北京
- **华南**：三亚、桂林
- **西南**：成都、重庆、丽江、大理、拉萨
- **西北**：西安、敦煌
- **东北**：哈尔滨、呼伦贝尔
- **华中**：张家界

## 🛠️ 技术栈

- **框架**：React 19
- **构建工具**：Vite 7
- **样式**：Tailwind CSS 4
- **动画**：Canvas API + CSS Transitions

## 📁 项目结构


## 🎨 核心功能说明

### 1. 转盘抽取
- 使用 Canvas API 绘制动态转盘
- 支持平滑的旋转动画（4秒，先快后慢）
- 自动适配城市数量，动态分配扇区

### 2. 筛选功能
- **地区筛选**：华东、华北、华南、西南、西北、东北、华中
- **类型筛选**：古镇、海滨、山岳、都市、草原、沙漠
- 实时更新转盘内容

### 3. 结果展示
展示城市的完整信息：
- 📍 地理位置和特色
- 🏷️ 城市类型标签
- ⭐ 推荐景点
- 🍜 特色美食
- 🌡️ 最佳旅游季节
- 💰 预算参考

## 📝 数据格式

城市数据存储在 `src/data/cities.json`，每个城市包含：

```json
{
  "id": 1,
  "name": "杭州",
  "province": "浙江",
  "region": "华东",
  "types": ["古镇", "都市"],
  "description": "人间天堂，西湖美景甲天下",
  "attractions": ["西湖", "灵隐寺", "宋城"],
  "bestSeason": ["春季", "秋季"],
  "budget": "中等",
  "highlights": "江南水乡风情，现代化都市",
  "foodRecommend": ["西湖醋鱼", "东坡肉"]
}
```
