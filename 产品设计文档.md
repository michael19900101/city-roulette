# 中国城市旅游转盘 - 产品原型设计文档

## 📋 文档信息
- **产品名称**：城市轮盘 / 去哪玩转盘
- **版本**：v1.0 MVP
- **设计日期**：2026-02-14
- **设计目标**：一个有趣的、随机推荐中国旅游城市的Web应用

---

## 🎯 产品目标

### 用户痛点
- 旅游目的地选择困难
- 总是去同样的热门城市
- 缺乏探索新城市的动力

### 解决方案
通过趣味化的转盘抽奖方式，让用户发现新的旅游目的地，增加决策的趣味性和随机性。

## 🎨 详细功能模块设计

### 模块 1：筛选条件区

**位置**：页面上部，转盘上方

**字段设计**：

| 筛选项 | 选项内容 | 默认值 |
|--------|---------|--------|
| 地区筛选 | 全部/华东/华北/华南/西南/西北/东北/华中 | 全部 |
| 城市类型 | 全部/古镇/海滨/山岳/都市/草原/沙漠 | 全部 |

**交互**：
- 初始状态：折叠（节省空间）
- 点击"筛选"按钮展开
- 选择后实时更新转盘内容
- "重置筛选"恢复到全部城市

---

### 模块 2：转盘区域（核心）

**视觉设计**：
```
转盘样式：
- 形状：正圆形
- 大小：直径 400-500px（响应式）
- 分区：动态分区（根据筛选后的城市数量）
- 颜色：每个分区不同颜色（渐变色）
- 文字：城市名称 + 小图标

指针：
- 位置：转盘正上方/正中心
- 样式：三角形箭头
- 颜色：醒目（红色/金色）
```

**转盘内容**：
- 每个扇区显示一个城市名称
- 城市按热门程度或随机排列
- 建议显示 8-12 个城市（太多会太小）

**动画效果**：
```
1. 点击"开始抽取"
2. 转盘开始旋转（加速）
3. 旋转 3-5 秒
4. 逐渐减速
5. 停在某个城市（缓动函数：ease-out）
6. 轻微回弹效果
7. 触发结果展示
```

---

### 模块 3：抽取按钮

**视觉**：
- 大按钮（明显、易点击）
- 渐变色背景
- 阴影效果
- 图标：🎲 骰子

**文案变化**：
- 初始："开始抽取"
- 旋转中："抽取中..."（禁用状态）
- 结束后："再抽一次"

**状态**：
- 默认：可点击
- 旋转中：禁用
- 有结果后：变为"再抽一次"

---

### 模块 4：结果展示区

**显示时机**：转盘停止后

**内容结构**：
```
┌─────────────────────────────────┐
│ 🎉 标题：恭喜！你的目的地是：      │
│                                 │
│ 城市名称（大字号、加粗）           │
│                                 │
│ 📍 位置：省份 + 地区特征          │
│ 🏷️  类型：城市类型标签            │
│ ⭐ 推荐景点：3-5个热门景点         │
│ 🌡️  季节推荐：适合XX季节游玩       │
│ 💰 预算参考：经济/中等/高端        │
│                                 │
│ [查看详情] [再抽一次] [分享结果]   │
└─────────────────────────────────┘
```

**动画效果**：
- 淡入效果
- 可选：烟花/彩带动画（庆祝感）

## 🔄 用户交互流程

### 主流程
```
用户进入页面
    ↓
[可选] 设置筛选条件
    ↓
点击"开始抽取"
    ↓
转盘旋转动画（3-5秒）
    ↓
停在某个城市
    ↓
展示城市详细信息
    ↓
用户选择：
    - 查看详情（弹窗/新页面）
    - 再抽一次（回到第3步）
    - 分享结果（社交媒体）
```

### 异常流程
- 筛选后城市<5个：提示"筛选条件太严格，请重新选择"
- 网络错误：显示友好提示

## 🎨 UI/UX 设计建议

### 配色方案
**选项 A：清新活力**
- 主色：#FF6B6B（珊瑚红）
- 辅色：#4ECDC4（青绿色）
- 背景：#F7F7F7（浅灰）
- 转盘：彩虹渐变色

**选项 B：中国风**
- 主色：#C41E3A（中国红）
- 辅色：#FFD700（金色）
- 背景：#FFF8DC（米色）
- 转盘：传统配色（红、金、青）

### 字体
- 标题：思源黑体 / PingFang SC Bold
- 正文：PingFang SC / Helvetica Neue

### 动画时长
- 转盘旋转：3-5秒
- 结果展示：0.5秒淡入
- 按钮反馈：0.2秒

---

## 📱 响应式设计

### 桌面端（>1024px）
- 转盘直径：500px
- 布局：单列居中
- 结果展示：卡片形式

### 平板（768px - 1024px）
- 转盘直径：400px
- 筛选条件：两列布局

### 移动端（<768px）
- 转盘直径：80vw（响应式）
- 筛选条件：折叠式
- 结果展示：全屏卡片

## 💡 创新点思考

### 可选增值功能
1. **"命中注定"模式**：根据用户当前位置推荐附近城市
2. **"穷游/土豪"模式**：根据预算筛选
3. **"跟着季节走"**：只显示当前季节适合的城市
4. **"盲盒模式"**：隐藏城市名，抽到后才显示（增加惊喜感）
5. **社交功能**：看看别人都抽到了哪里
